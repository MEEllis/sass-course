## 在gulp 使用 sass 教程

sass基于Ruby语言开发而成，因此安装sass前需要安装Ruby。
建议安装rubu的时候直接选择安装在c盘系统盘

 先安装 ruby , 再通过
 ```
 gem install sass
 gem install compass
 ```
 安装 sass ， compass
 不清楚的，具体可以看这篇文章 [如何配置sass环境](https://www.sass.hk/install/).
  
## Clone sass教程

  ```
         git clone https://github.com/MEEllis/sass-course.git
         cd sass-course
  ```


 第一步：
   ```
     npm install
  ```
 二步：(执行gulp默认的任务)
  ```
  gulp
 ```
